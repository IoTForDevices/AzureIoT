# AzureIoT
Collection of articles and samples explaining different Azure IoT Services.

- Samples/IoTHub contains scripts to create X.509 root/intermediate/device certificates to be used with IoT Hub. It also contains sample code to verify the generated certificate chain by a simulated device. Take a look at the [IoTHub-X509-Certs explanation](Samples/IoTHub/Scenario/IoTHub-X509-Certs/README.md) to get started.
